Smart Parking Management System

Developed a Parking Management System integrated with an Android Application that provides real time status of nearest parking slots to the user and allows them to reserve the slot. In addition it can also display the route to the user. Utilized Ultraviolet sensors to detect occupancy and Google Firebase for updating real time status of the slots. Google Places and Maps API used to gather the locations.

Designed the circuit using Raspberry Pi,RFID Tags and other sensors. Created a flow diagram to explain various scenarios. Programmed the application in Arduino and presented it to our college and ready to implement after the college reopens.
